# 血压记录应用使用说明

## 项目概述
这是一个基于原生Android (Kotlin)开发的血压记录应用，具有以下功能：
- 记录血压和心率数据
- 显示历史记录列表
- 展示血压变化趋势图表
- 界面美观，操作简单

## 项目结构
项目采用MVVM架构，包含以下主要组件：
- **数据层**：Room数据库存储血压记录
- **仓库层**：处理数据访问逻辑
- **ViewModel层**：管理UI状态和业务逻辑
- **UI层**：用户界面组件

## 如何构建和运行

### 方法一：使用Android Studio（推荐）
1. 下载并安装最新版本的Android Studio
2. 打开Android Studio，选择「Open an existing project」
3. 导航到项目目录 `c:\Users\Administrator\Desktop\222\血压记录应用` 并打开
4. 等待Gradle同步完成
5. 点击「Run」按钮或使用快捷键 `Shift+F10` 运行应用

### 方法二：使用命令行
1. 确保已安装JDK 8或更高版本
2. 确保已安装Android SDK
3. 打开命令行，导航到项目目录
4. 运行以下命令构建APK：
   ```
   ./gradlew assembleDebug
   ```
5. 构建完成后，APK文件将位于 `app/build/outputs/apk/debug/app-debug.apk`

## 应用功能说明

### 主界面
- 显示最近的血压记录
- 提供「添加记录」和「查看历史」按钮

### 添加记录
- 输入收缩压、舒张压和心率
- 自动显示血压水平（正常/高血压/低血压）
- 可添加备注信息

### 历史记录
- 列表视图：显示所有记录，可滑动查看
- 图表视图：展示血压和心率的变化趋势
- 点击记录可删除

## 技术栈
- **开发语言**：Kotlin
- **架构**：MVVM
- **数据库**：Room
- **UI**：Material Design
- **图表**：MPAndroidChart

## 注意事项
- 本应用仅用于记录和展示血压数据，不能替代专业医疗建议
- 请在医生指导下使用本应用
- 定期备份您的血压数据
